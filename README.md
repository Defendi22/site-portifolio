# 🚀 Currículo Premium Interativo - Fernando Defendi

Um portfólio digital moderno, premium e altamente interativo desenvolvido com **React**, **TypeScript** e **Tailwind CSS** no estilo **glassmorphism**.

## ✨ Características Principais

### 🎨 Design Moderno
- **Glassmorphism Premium**: Design elegante com efeito vidro transparente
- **Gradientes Dinâmicos**: Paleta de cores moderna com blue, purple e pink
- **Animações Suaves**: Transições fluidas com Framer Motion
- **Responsivo**: Completamente adaptado para desktop, tablet e mobile
- **Dark Mode**: Interface escura premium similar aos sites de Apple, Stripe e Linear

### 🎬 Animações Avançadas
- **Parallax Hero**: Efeito de profundidade na seção hero
- **Scroll Reveal**: Elementos aparecem animados ao rolar a página
- **Hover 3D**: Cards com efeito 3D ao passar o mouse
- **Partículas Interativas**: Fundo com efeito de partículas animadas
- **Contadores Animados**: Números que contam até o valor final
- **Navbar Fixa**: Barra de navegação com glassmorphism que acompanha o scroll

### 📱 Seções Incluídas
1. **Hero Section**: Apresentação profissional com foto de perfil e CTA buttons
2. **About Me**: Sobre mim com highlights de habilidades
3. **Stats**: Contadores animados de experiência e projetos
4. **Tecnologias**: Cards com stack completo de ferramentas
5. **Experiência**: Timeline interativa de experiência profissional
6. **Projetos**: Galeria de projetos com hover effects
7. **FAQ**: Acordeão com perguntas frequentes
8. **Contato**: Formulário moderno e informações de contato
9. **Footer**: Rodapé com links sociais

### 🎯 Funcionalidades Especiais
- ✅ **SEO Otimizado**: Meta tags, estrutura semântica
- ✅ **Acessibilidade**: ARIA labels, navegação por teclado
- ✅ **Performance**: Código otimizado, lazy loading
- ✅ **Mobile First**: Design responsivo e intuitivo
- ✅ **Formulário de Contato**: Com validação e feedback visual
- ✅ **Social Links**: Integração com LinkedIn, GitHub e WhatsApp
- ✅ **Scroll Smooth**: Navegação suave entre seções

## 🚀 Como Usar

### Instalação
```bash
# Clonar o repositório
git clone <seu-repositorio>
cd portfolio

# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview do build
npm preview
```

### Estrutura do Projeto
```
src/
├── components/
│   ├── Navbar.tsx           # Barra de navegação fixa
│   ├── Hero.tsx             # Seção hero com parallax
│   ├── About.tsx            # Sobre mim
│   ├── Stats.tsx            # Estatísticas com contadores
│   ├── Technologies.tsx      # Stack de tecnologias
│   ├── Experience.tsx        # Timeline de experiência
│   ├── Projects.tsx          # Galeria de projetos
│   ├── FAQ.tsx              # Perguntas frequentes
│   ├── Contact.tsx          # Formulário e contato
│   ├── Footer.tsx           # Rodapé
│   └── ParticleBackground.tsx # Fundo com partículas
├── hooks/
│   └── useScrollReveal.ts   # Hook para scroll reveal
├── App.tsx                  # Componente raiz
├── main.tsx                 # Entry point
├── index.css               # Estilos globais
├── index.html              # HTML principal
└── vite.config.ts          # Configuração Vite
```

## 🔧 Customização

### Alterar Cores
Edite as cores no `index.css` ou use classes Tailwind para customizar o design.

### Adicionar Conteúdo
Todos os textos e dados estão nas variáveis dos componentes. Basta editar:
- `hero.tsx` - Informações pessoais
- `about.tsx` - Sobre você
- `technologies.tsx` - Skills
- `experience.tsx` - Experiência profissional
- `projects.tsx` - Portfólio

### Adicionar Imagem de Perfil
Substitua `/public/avatar.jpg` pela sua imagem pessoal.
